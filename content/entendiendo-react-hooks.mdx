# Entendiendo los React Hooks / Understanding React Hooks

Los React Hooks revolucionaron la forma en que escribimos componentes en React, permitiéndonos usar estado y otras características de React sin escribir una clase.

React Hooks revolutionized the way we write components in React, allowing us to use state and other React features without writing a class.

## ¿Por qué Hooks? / Why Hooks?

Antes de los Hooks, las características como el estado y el ciclo de vida solo estaban disponibles en componentes de clase. Esto llevaba a:

Before Hooks, features like state and lifecycle were only available in class components. This led to:

*   **Lógica compleja en componentes de clase:** Dificultad para reutilizar lógica con estado.
*   **`this` confuso:** El contexto de `this` en JavaScript puede ser complicado.
*   **Componentes funcionales limitados:** No podían tener estado.

### `useState`

El Hook `useState` te permite añadir estado a componentes funcionales.

The `useState` Hook lets you add state to functional components.

```jsx
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```

### `useEffect`

El Hook `useEffect` te permite realizar efectos secundarios en componentes funcionales. Es un reemplazo para `componentDidMount`, `componentDidUpdate` y `componentWillUnmount`.

The `useEffect` Hook lets you perform side effects in functional components. It is a replacement for `componentDidMount`, `componentDidUpdate`, and `componentWillUnmount`.

```jsx
import React, { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `You clicked ${count} times`;
  }, [count]); // Only re-run the effect if count changes

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```

## Otros Hooks / Other Hooks

React también proporciona otros Hooks como `useContext`, `useReducer`, `useCallback`, `useMemo`, `useRef`, etc., cada uno con su propósito específico para optimizar y simplificar el código.

React also provides other Hooks like `useContext`, `useReducer`, `useCallback`, `useMemo`, `useRef`, etc., each with its specific purpose to optimize and simplify code.

## Conclusión / Conclusion

Los Hooks son una adición poderosa a React que promueve la reutilización de lógica con estado y hace que los componentes funcionales sean más capaces.

Hooks are a powerful addition to React that promotes stateful logic reuse and makes functional components more capable.
